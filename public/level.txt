# HOW TO EDIT THIS FILE:
#   Each character below represents one 48x48 pixel tile.
#   Add/remove characters to resize the level.
#   Lines starting with # are comments (ignored).
#   Blank lines are ignored.
#   The camera follows the player, so levels can be any size!
#
# TILE LEGEND:
#   .   Air (empty space)
#   #   Solid block (walls, ceilings — blocks from all sides)
#   -   Platform (thin — stand on top, jump through from below)
#   =   Ground (circus-striped floor)
#   P   Player spawn (where new players appear — acts as air)
#   @   Bounce pad (launches players upward!)
#   *   Balloon decoration (floating, no collision)
#   ~   Confetti decoration (sparkly, no collision)
#
# TIPS:
#   • Put = at the bottom so players don't fall into the void
#   • Place P where you want players to spawn (above a surface!)
#   • Solid # blocks stop movement from ALL directions
#   • Platforms - only block from above (jump up through them)
#   • Bounce pads @ are fun on the ground or on platforms
#   • The level below is 42 wide x 20 tall = 2016 x 960 pixels
#   • Try making your own levels — it's just text!
